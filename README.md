# Retro Travel Postcard Collection

Digitize a retro travel postcard collection into a modern web app. Users can
save postcards, see them on a map, and manage their local collection.

[Review Initial Planning Document](https://docs.google.com/document/d/1d8EHrMZNiadDDqU_kKqquoMQ7JJkDojNZjGZeH3T9nA/edit?tab=t.0#heading=h.6d8jrn8te4io)

## Tech Stack

- Next.js (Pages Router) with TypeScript
- Tailwind CSS
- Leaflet + React-Leaflet 
- LocalStorage for data persistence

## Getting Started

```bash
npm install
npm run dev
